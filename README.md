# EUA Afora (Around The U.S.)

Projeto 17 da Tripleten. Site em React: cadastro, login, perfil e cartões de lugares. O login fica salvo no navegador até você sair.

## Como abrir o projeto

```
npm install
npm run dev
```

Depois abra o link que aparecer no terminal (em geral `http://localhost:5173`).

Comando para build: `npm run build`

## Tecnologias

React, Vite, CSS (BEM), React Router. APIs em `src/utils`.
